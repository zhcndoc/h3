# Cookie

> H3 cookie 工具。

<!-- automd:jsdocs src="../../src/utils/cookie.ts" -->

### `deleteCookie(event, name, serializeOptions?)`

通过名称删除 cookie。

### `getCookie(event, name)`

通过名称获取 cookie 值。

### `parseCookies(event)`

解析请求以获取 HTTP Cookie 头字符串，并返回所有 cookie 名称-值对的对象。

### `setCookie(event, name, value, options?)`

通过名称设置 cookie 值。
