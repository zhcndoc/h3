  • 运行时无关

    无缝兼容任何 JavaScript 运行时，包括 Deno、Bun、Node.js、Workers 等。基于 Web 标准，并由 Srvx 提供支持。

  • 微小、可组合且快速

    具有低延迟和最小的捆绑影响，设计时考虑了可组合性和最大兼容性。

  • 人体工学与安全的 API

    精心设计、符合人体工学且强类型的 API，内置数十种可摇树优化的实用工具。
